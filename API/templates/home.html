send post req to train_predict

If trainer is already trained on the named data (pandas df as csv) but uniquely identified by caller It uses the existing trained model with the same name to infer on the posted data

request.files: the files in the body, which Flask keeps separate from form. HTML forms must use enctype=multipart/form-data or files will not be uploaded.

Header args:
force_retrain = Defaults to NO. If enabled, it will overwrite the previous training of the same train file
epochs = Defaults to 10. If set, will train for given epochs. But only if needed or forced.
